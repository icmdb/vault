# valut - hashicorp

## Quick Start

```
# Clone Repo
git clone https://github.com/icmdb/vault.git

# Enter Directory
cd vault

# Build
make build
#   Or
make cnbuild    # For China Mainland

# Copy Binary to valut-${GOOS}
make run
```
